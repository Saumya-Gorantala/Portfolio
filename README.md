# Saumya Gorantala — Portfolio

A personal portfolio website built with React, Vite, TypeScript, and Tailwind CSS.

## Tech Stack

- **Framework**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **UI Components**: Radix UI / shadcn/ui

## Getting Started

```bash
# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build
```

## Deployment

Deploy easily to [Netlify](https://netlify.com) or [Vercel](https://vercel.com) by connecting your GitHub repository.
